Introducción a HTML

Lenguaje de Marcado de Hipertexto

¿Qué significa HTML?

HTML significa HyperText Markup Language

HyperText: Texto que contiene enlaces a otros textos

Markup: Marcado o etiquetado

Language: Lenguaje

Es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de un sitio web.

Breve Historia del HTML

1991: Tim Berners-Lee crea HTML en el CERN

1995: HTML 2.0 - Primera especificación estándar

1997: HTML 3.2 y 4.0 - Más funcionalidades

2014: HTML5 - Versión actual, con soporte multimedia

HTML evoluciona constantemente para adaptarse a las necesidades de la web moderna.

¿Por qué es un Lenguaje de Etiquetas?

HTML usa etiquetas (tags) para marcar y definir diferentes tipos de contenido.

Las etiquetas le dicen al navegador:

  • "Esto es un título"
  • "Esto es un párrafo"
  • "Esto es un enlace"
  • "Esto es una imagen"

No es un lenguaje de programación, sino de marcado: describe la estructura del contenido.

Tipos de Etiquetas

1. Etiquetas de Par (Apertura y Cierre)

<p>Este es un párrafo</p>
<h1>Este es un título</h1>

2. Etiquetas Vacías (Solo apertura)

<img src="imagen.jpg" alt="Descripción">
<br>
<hr>

El contenido visible es lo que queda entre las etiquetas de apertura y cierre.

Estructura Base de un Documento HTML

<!DOCTYPE html>
<html lang="es">
<head>
<!-- Metadatos aquí -->
</head>
<body>
<!-- Contenido visible aquí -->
</body>
</html>

<!DOCTYPE html> - Declara que es un documento HTML5

<html> - Raíz del documento

<head> - Información sobre el documento

<body> - Contenido visible de la página

¿Qué es <head>?

El <head> contiene metadatos sobre el documento.

Esta información NO se muestra en la página, pero es importante para:

  • El navegador (cómo interpretar la página)
  • Los buscadores (SEO)
  • Definir el título de la pestaña
  • Vincular archivos CSS y JavaScript
  • Definir la codificación de caracteres

Ejemplo de <head> Básico

<head>
<meta charset="UTF-8">
<meta name="viewport"
content="width=device-width, initial-scale=1.0">
<meta name="description"
content="Mi primera página web">
<title>Mi Sitio Web</title>
</head>

charset: Codificación de caracteres (UTF-8 incluye ñ, tildes, etc.)

viewport: Para diseño responsive

title: Título que aparece en la pestaña del navegador

¿Qué es <body>?

El <body> contiene todo el contenido visible de la página web.

Aquí va todo lo que el usuario verá:

  • Textos, títulos y párrafos
  • Imágenes y videos
  • Enlaces y botones
  • Tablas y formularios
  • Cualquier contenido interactivo

Ejemplo de <body> Básico

<body>
<h1>Bienvenido a Mi Sitio</h1>
<p>Este es un párrafo de ejemplo.</p>

<h2>Sobre mí</h2>
<p>Me gusta programar en HTML.</p>

<a href="contacto.html">Contáctame</a>
</body>

Todo esto se verá en la pantalla del navegador.

Atributos y Valores

Los atributos proporcionan información adicional sobre las etiquetas.

<etiqueta atributo="valor">contenido</etiqueta>

Ejemplos:

<a href="https://ejemplo.com">Enlace</a>
<img src="foto.jpg" alt="Descripción">
<p id="principal" class="texto">Párrafo</p>

href, src, alt, id, class son atributos

Los valores van siempre entre comillas

Etiquetas de Texto y Navegación

Texto:

<h1> a <h6> - Encabezados (h1 más grande)
<p> - Párrafos
<strong> - Texto en negrita
<em> - Texto en cursiva
<br> - Salto de línea

Navegación:

<a href="url">Texto del enlace</a>
<nav> - Sección de navegación
<ul> y <li> - Listas no ordenadas
<ol> y <li> - Listas ordenadas
